AWS 服务商介绍

简单的介绍

AWS 服务商是亚马逊旗下的云服务器服务商，其主要特点如下

  • 优势
    • 服务器稳定可靠，大部分厂家都是用它
    • 国外支持的区域多
    • 新账号有 1 年的免费使用套餐
    • 网速没有带宽限制
    • 国内连接 AWS 服务器相对丝滑
  • 劣势
    • 相比于国内服务商价格偏贵
    • 新手初次使用极度不友好

付费

  • 每个月的前几号是账单支付，采用信用卡扣款方式
  • 如果信用卡扣款失败，则会封号处理

风控

AWS 风控及其严格，任何一点的不注意就会导致封号。

  • 同一个 IP 多次登录账号
  • 新账号使用高配置机器
  • 新账号大量开启服务器
  • ...

一年免费套餐

AWS 新账号支持特定机型的免费套餐，通常是 1核1G，不同区域有例外，比如，香港地区的免费套餐，服务器是 2核1G。

免费套餐的细节如下

  • 一个月免费使用时常为 750 小时
  • 出流量一个月为 100G
  • 每个区域免费套餐的机型、镜像、存储都是固定的，相关的细节可以去 AWS 官网查找「开启实例的时候，支持免费套餐会在后面标明」

这里举一个东京的免费机型。

  • 镜像: Ubuntu 18.04 「免费套餐」
  • 服务器: t2.micro
  • 存储: 8G

其他

对于可能存在的问题，进行如下回答：

  • 问: 我创建了两个免费东京套餐机器，两个机器连续运转 10 天，是否产生费用
    • 答: 计算如下，假设，一个月就两台机器运行 10 天，则计算如下 2 * 10 * 24 = 480 小时，在流量和其他指标没超过的情况下，不进行计费。
  • 问: 我为什么不能开启香港区域的机器
    • 答: AWS 默认只会开启部分区域，如果，想要开启其他区域需要到 AWS 账号上进行启动
  • 问: 32v 和 8v 代表什么意思？
    • 答: v 代表核数，标明你这个账号可以开启多少核的机器。
